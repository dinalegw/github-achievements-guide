# Quickdraw

## Overview

Quickdraw rewards fast, helpful contributions that move a project forward quickly. It highlights responsiveness in issues, pull requests, and community triage.

## Requirements

* Respond quickly to one or more issues or pull requests.
* Offer a useful fix, review, or improvement within a short timeframe.
* Keep feedback actionable and concise.

## Step-by-step

1. Watch repositories that interest you.
2. Identify an issue or PR where you can help fast.
3. Add a comment with a clear path forward.
4. If appropriate, open a small PR or review an existing one.
5. Follow up quickly on any maintainer response.

## Example

* Spot an issue with outdated documentation.
* Comment with a proposed update and open a PR within hours.
* If someone requests changes, act fast to update the branch.

## Tips

* Use notifications or GitHub saved replies to work faster.
* Keep responses positive and focused.
* When possible, add a small code change rather than only commentary.
