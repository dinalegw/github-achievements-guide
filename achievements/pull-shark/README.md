# Pull Shark

## Overview

Pull Shark is awarded for making fast, consistent pull request contributions across public repositories. This achievement shows that you can identify problems, propose safe changes, and collaborate effectively.

## Requirements

* Open or review at least 10 pull requests.
* Use at least 3 different public repositories.
* Provide useful PR descriptions and follow-up responses.

## Step-by-step

1. Choose active projects with clear contribution guidelines.
2. Start with documentation fixes, small bugs, or tests.
3. Open a pull request with a concise title and summary.
4. Respond to review feedback and update your PR promptly.
5. Repeat until you have 10 meaningful PR contributions across repositories.

## Example

* Find a repo with a `good first issue` label.
* Fix a typo in docs and open a PR.
* Submit a second PR to add a small test or improve an error message.
* Track progress across repositories so each PR counts.

## Tips

* Keep each PR small and focused.
* Reference related issues in your PR description.
* Use labels or GitHub issue templates when available.
