# Pair Extraordinaire

## Overview

This achievement recognizes meaningful collaboration with another contributor in a GitHub repository.

## How to unlock

1. Collaborate on a feature, issue, or review with another contributor.
2. Share progress through comments and updates.
3. Land the contribution in a public repository.

## Tips

* Use GitHub discussions or issue comments to coordinate work.
* Pair on a live coding session, design review, or issue fix.
