# Contributing to GitHub Achievements Guide

Thank you for helping improve this project! We welcome contributions of any size, from typo fixes to new achievement guides.

## How to Contribute

1. Fork the repository.
2. Create a branch for your work.
3. Add or update content in the `docs/`, `achievements/`, or `examples/` folders.
4. Open a pull request with a clear description of your changes.

## Content Guidelines

* Keep language friendly and concise.
* Use headings, bullet points, and examples.
* Link to related sections when helpful.
* Keep achievement instructions accurate and up to date.

## Code and Website Changes

* Run any website preview or static site build before submitting.
* Keep JavaScript minimal and accessible.
* Use semantic HTML for the website.

## Reporting Issues

* Open an issue for outdated information or missing achievement details.
* Provide reproduction steps for bugs in the website or docs.
