# Pull Shark

## Overview

Pull Shark is awarded for opening or reviewing the required number of pull requests in public repositories.

## How to unlock

1. Find an open-source repo that welcomes contributors.
2. Open or review 10 pull requests across at least 3 repositories.
3. Keep your contributions meaningful and documented.

## Tips

* Start with documentation fixes or bug reports.
* Use issue labels like "good first issue".
* Keep PRs small and reviewable.
